# Funnel Chart

A Funnel Chart in REAVIZ is perfect for visualizing the flow of data through various stages of a process. 
Each slice represents a step, with the final slice showing the end result. 
This type of chart helps you understand conversion rates and process efficiency.

### Where to use:
- **Track Conversion Rates:** Ideal for visualizing conversion rates at each stage.
- **Analyze Processes:** Useful for identifying drop-offs in multi-step processes.
- **Evaluate Workflow Efficiency:** Great for assessing the efficiency of a workflow.

## Quick Start
To create a funnel chart, use import the `FunnelChart` and give it some `data`.


<Runner code={`import { FunnelChart } from 'reaviz';

export default function App() {
  return (
    <FunnelChart
      height={300}
      width={400}
      data={[
        {
          data: 1000,
          key: 'Visited Site'
        },
        {
          data: 900,
          key: 'Added to Cart'
        },
        {
          data: 600,
          key: 'Initiated Checkout'
        }
      ]}
    />
  );
}`}
/>
