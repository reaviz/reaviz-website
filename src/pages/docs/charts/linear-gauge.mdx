# Linear Gauge

Linear Gauge uses a linear scale to display a specific data point using a dial over a linear scale with defined limits.

REAVIZ supports:

- [Single-series gauges](/docs/charts/linear-gauge#single-series-gauges)
- [Multi-series gauges](/docs/charts/linear-gauge#multi-series-gauges)

## Quick Start
To create a Linear Gauge, use import the `LinearGauge` and give it `data`. The chart
will automatically configure itself with the default options exposed via `props`.

<Runner code={`import { LinearGauge } from 'reaviz';

export default function App() {
  return (
    <LinearGauge
      width={300}
      height={100}
      data={{ key: 'Risk Score', data: 74 }}
    />
  );
}`}
/>

In this example, we only pass one data object, however, we can pass multiples and it will
distribute the values across them.
