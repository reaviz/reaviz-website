# Radial Gauge

Radial Gauge uses a radial scale to display a specific data point using a dial over a radial scale with defined limits.

Types supported by reaviz:

- [Single](/docs/charts/radial-gauge#single)
- [Stacked](/docs/charts/radial-gauge#stacked)

## Quick Start
To create a Radial Gauge, use import the `RadialGauge` and give it `data`. The chart
will automatically configure itself with the default options exposed via `props`.

<Runner code={`import { RadialGauge } from 'reaviz';

export default function App() {
  return (
    <RadialGauge
      height={300}
      width={300}
      data={[{ key: 'Austin, TX', data: 75 }]}
    />
  );
}`}
/>

In this example, we only pass one data object, however, we can pass multiples and it will
distribute the values across them.
