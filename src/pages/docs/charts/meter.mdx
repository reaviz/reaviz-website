# Meter

Meter chart is a visual to show a single value on a meter scale which is in this cases is a linear gauge like chart divided in sections.
It is designed to indicate a single value within a set range or target threshold.

## Quick Start
To create a meter, use import the `Meter` and give it `value` point.

<Runner code={`import { Meter } from 'reaviz';

export default function App() {
  return (
    <Meter value={50} style={{ width: 300 }} />
  );
}`}
/>
