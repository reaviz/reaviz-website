# Line Chart

A Line Chart in REAVIZ depicts data using points connected by lines, emphasizing trends and changes over a continuous set of values, 
typically along an x-axis and a y-axis. Line charts are effective for visualizing data trends over time or continuous data points.

### Types supported by reaviz:

- [Single Series](/docs/charts/line-chart#single-series)
- [Multi Series](/docs/charts/line-chart#multi-series)
- [Radial](/docs/charts/line-chart#radial)
- [Stacked](/docs/charts/line-chart#stacked)
- [Stacked Normalized](/docs/charts/line-chart#stacked-normalized)

### Where to use:

- **Show Trends Over Time:** Ideal for visualizing how data changes over a period.
- **Identify Patterns:** Useful for identifying patterns or trends in data sets.
- **Compare Multiple Data Sets:** Great for comparing multiple sets of data.
- **Track Changes:** Effective for tracking changes in data points.

## Quick Start
To create a line chart, use import the `LineChart` and give it `data`. The chart
will automatically configure itself with the default options exposed via `props`.

<Runner code={`import { LineChart } from 'reaviz';

export default function App() {
  return (
    <LineChart
      height={300}
      width={300}
      data={[
        { key: new Date('11/29/2019'), data: 13 },
        { key: new Date('11/30/2019'), data: 23 },
        { key: new Date('12/1/2019'), data: 7 },
      ]}
    />
  );
}`}
/>
