A radar chart is a graphical method of displaying multivariate
data in the form of a two-dimensional chart of three or more quantitative
variables represented on axes starting from the same point. The relative
position and angle of the axes is typically uninformative.

## Quick Start
To create a radar, use import the `RadarChart` and give it `value` point.

```jsx
import { RadarChart } from 'reaviz';

const data = [
  {
    key: 'Lateral Movement',
    data: [
      {
        key: 'XML',
        data: 0
      },
      {
        key: 'JSON',
        data: 120
      },
      {
        key: 'HTTPS',
        data: 150
      }
    ]
  },
  {
    key: 'Discovery',
    data: [
      {
        key: 'XML',
        data: 100
      },
      {
        key: 'JSON',
        data: 34
      },
      {
        key: 'HTTPS',
        data: 0
      }
    ]
  }
];

const MyChart = () => (
  <RadarChart data={data} height={450} width={450} />
);
```
