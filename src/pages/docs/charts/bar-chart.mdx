# Bar Chart
A bar chart uses rectangular bars to show values, with one axis for categories and the other for values. 
It's great for visualizing data like months, age groups, or product types. 
Bar charts help compare different categories and track trends over time. 
With REAVIZ, you can easily create and customize bar charts to suit your data visualization needs!

### Types supported by reaviz:

- Single Series [Vertical](/docs/charts/bar-chart#single-series-vertical) / [Horizontal](/docs/charts/bar-chart#single-series-horizontal)
- Multi Series [Vertical](/docs/charts/bar-chart#multi-series-vertical) / [Horizontal](/docs/charts/bar-chart#multi-series-horizontal)
- Stacked [Vertical](/docs/charts/bar-chart#stacked-vertical) / [Horizontal](/docs/charts/bar-chart#stacked-horizontal)
- Stacked Normalized [Vertical](/docs/charts/bar-chart#stacked-normalized-vertical) / [Horizontal](/docs/charts/bar-chart#stacked-normalized-horizontal)
- Stacked Diverging [Vertical](/docs/charts/bar-chart#stacked-diverging-vertical) / [Horizontal](/docs/charts/bar-chart#stacked-diverging-horizontal)
- [Marimekko](/docs/charts/bar-chart#marimekko)
- [Sparkline](/docs/charts/bar-chart#sparkline)
- [Waterfall](/docs/charts/bar-chart#waterfall)
- [Histograms](/docs/charts/bar-chart#histograms)

### Where to use:
- **Compare Quantities:** Compare different categories, like sales figures.
- **Highlight Largest/Smallest Categories:** Identify categories with highest or lowest values.
- **Visualize Categorical Data:** Display distinct categories, such as age groups or product types.

## Quick Start
To create a Bar chart, use import the `BarChart` and give it `data`. The chart
will automatically configure itself with the default options exposed via `props`.

<Runner code={`import { BarChart } from 'reaviz';

export default function App() {
  return (
    <BarChart
      height={300}
      width={300}
      data={[
        { key: 'DLP', data: 13 },
        { key: 'SIEM', data: 2 },
        { key: 'Endpoint', data: 7 }
      ]}
    />
  );
}`}
/>

### Customization

Customize the Bar Chart using various props:

- **colorScheme:** Array of colors for the chart.
- **gridlines:** Toggle gridlines on or off.
- **bar:** Customize the appearance of the bars.
- **legend and tooltips:** Add legends and tooltips for better data interpretation.
