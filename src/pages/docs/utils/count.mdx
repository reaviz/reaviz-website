# Count

Count from one number to another.

## Quick Start
To use the Count component, use import the `Count` and give it a number `to`. Optionally
you can give the `from` number to start from.

<Runner code={`import { Count } from 'reaviz';

export default function App() {
  return (
    <Count
      from={0}
      to={50}
    />
  );
}`}
/>
