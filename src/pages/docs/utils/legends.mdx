# Legends

The legend component is used to provide a visual representation of the data series in a chart. There are two types of legends in reaviz:

- Discrete - Out of the box, none of the charts include legends because we want to keep the library as modular as possible.
- Sequential - The sequential legend is useful for categorical charts such as heatmaps. This legend supports supports both vertical and horizontal layout.
